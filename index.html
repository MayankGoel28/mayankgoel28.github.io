The Domain-Specific Search Engine is a search engine modelled on Walmart data. It uses semantic distance and some other weights to make an efficient search on a Knowledge-Based graph, generated by Clustering.

Jeeves is a joke generator, using templates generated from data, and then filling them using relevant words. It was part of a research project that was accepted to the 11th Humor Research Conference.

The Summarizer Bot uses cosine similarity on sentence vectors to summarize long texts, along with keyword extraction for a discord bot. Encountered and solved problems such as code mixing and data processing using a predominant language agnostic approach.

Dr Bear is a real-time scraper which collects news on social media opinion on Stocks from multiple sources. The scraper was used as part of a larger webapp for usage to survey opinions on stocks and companies at even given time.
